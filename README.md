# Open Source HTML Tetris Game

This project is a browser-based, open-source Tetris game built with HTML. 

## Features
* *User-Friendly:* Simple and intuitive gameplay.
* *Turkish Language Support:* Fully localized in Turkish.
* *In-Game Stats:* Real-time score tracking.
* *Next Piece Preview:* Displays the upcoming tetromino in the top-right corner.
* *Easy Controls:* Gameplay instructions are clearly listed in the bottom-left corner.

## Usage
To play the game, simply open the index.html file in any modern web browser.

## Tech Stack
* HTML5
* CSS3
* JavaScript
